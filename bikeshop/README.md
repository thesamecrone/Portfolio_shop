# Veloretti Bikeshop

Veloretti Bikeshop is a fully responsive e-commerce project for bicycles, demonstrating both frontend and backend development skills. The project features complex JavaScript logic, backend integration with Node.js/Express.js, and PostgreSQL database operations.

## Features

- Product catalog with dynamic variants (color, model)
- Interactive product modifications implemented in pure JavaScript
- Carousel sliders for products
- Page translation (English/Russian)
- Email subscription form for newsletters
- Google authentication for login
- AOS animations for smooth scrolling effects
- Fully responsive design for desktop, tablet, and mobile

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: PostgreSQL
- Authentication: Google OAuth 2.0
